## **常用的正则表达式**

* 检测有效数字
    - 正数 负数 零
    - 小数 整数
    + 分析规则
        - 可以出现 +/1，可以没有，也可以有一个
        - 整数 0 12 9 一位或者多位数字，一位0 -9 多位数字不能以0开头
        - 小数部分
    ``` javascript
    let reg = /^[+-]?(\d|([1-9]\d))(\.\d+)?$/
    ```


* 检测电话号码

```javascript
let reg = /^1\d{10}$/
```

* 检测中文姓名
    - 中文汉字 【\u4E00 - \u9FA5】

let reg =  /^[\u4E00-\u9FA5]{2,}(·[\u4E00-\u9FA5]{2，})?$/

* 检测邮箱

let reg = /^\w+([-.]\w+)*@[A-Za-z0-9]+((\.|-)[A-Za-z0-9]+)*\.[A-Za-z0-9]+$/;
