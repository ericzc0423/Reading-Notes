##python数据类型

* 基本类型
    - 数字
        + 整数 int
        + 长整型 long
        + 泛点型 float
    - 字符串
        + 文本 str
        + 字节 bytes
    - 布尔
        + True
        + False
* 数据集
    - 列表 list
    - 元组 tuple
    - 字典 dict
        + 有序字典
        + 无序字典
    - 集合 set
        + 有序集合
        + 无序集合


`int整型`
- 32位，-2**32~2**31-1
- 64位，-2**63〜2*63-1
`python3 中不再有long`

`str 字符串`
- 加了引号的字符都被认为是字符串
- 单引号、双引号没有任何区别， 多引号 ''' ''' 多行字符的引用

* 字符串拼接，必须是双方都是字符，不能是数字或者是其它类型
